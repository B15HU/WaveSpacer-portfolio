### Lyhyt projektin kuvaus:

WaveSpacer on musiikkisovellus, jonka tavoitteena on auttaa käyttäjiä hallitsemaan suuria Spotify-soittolistojaan.
Sovelluksen tarkoituksena on tehdä musiikin kuuntelusta mielekkäämpää poistamalla turhat kappaleet soittolistoista. Tämä tapahtuu käyttäjäystävällisen käyttöliittymän ja soittolistojen sisällön selkeän visualisoinnin avulla.
Projektin tavoitteena oli toteuttaa toimiva sovelluskonsepti ideasta valmiiksi demoksi ja testata sen toimivuutta oikeilla käyttäjillä.

Toteutimme projektin noin 3,5 kk aikana 5hengen kehittäjä tiimissä. Vastuu alueenani oli toimia – **Testaajana & Liiketalous vastaavana.** Tässä portfoliossa esittelen TC-opintojakson aikana tekemäni työt, osaamiseni kehittymisen sekä omat oivallukseni opintojakson ajalta.


### Koodaaminen

Aloitin projektin parissa Frontend kehittäjänä, koska vastuu alueeni tehtäviä ei vielä ollut. Esituotannon aikana meidän UI/UX suunitelmien pohjalta jaettiin työt frontti devaajan kanssa hänen johdolla. Pääsin kirjoittamaan koodia tailwindillä, mikä oli tuttua hommaa Web Ui 2 -kurssilta. Sillä aikaan kun Frontti koodaaja teki etusivua pääsin työstämään kirjaudu ja rekisteröidy -näkymiä, myöhemmin tein myös soittolistan valitsemis ¨-näkymän sekä tervehdys animaation kun kirjautuu sisälle.
